calendarchy

A terminal calendar that shows your Google Calendar and iCloud events side by side.

Features

  • Google Calendar and iCloud Calendar in one view
  • Month grid with day-by-day event browsing
  • One-click join for Zoom, Google Meet, and Teams links
  • Accept or decline events with a keystroke
  • Offline-first with local event caching
  • Zero-config Google sign-in via browser
  • iCloud via app-specific password (CalDAV)
  • Native macOS EventKit support for iCloud

Install

Arch Linux (AUR)
yay -S calendarchy-bin
From source
git clone https://github.com/sovanesyan/calendarchy.git
cd calendarchy && cargo build --release

Keyboard

hPrevious day lNext day jNext week kPrevious week tJump to today oOpen meeting link aAccept event dDecline event /Search events SSetup wizard qQuit
